## 🧠 THE LOGIC (PLAY-BY-PLAY)
1. **The Call**: An app or the CLI sends a prompt to the Engine.
2. **The Strategy**: `KeyPool` picks the optimal API key based on the selected rotation strategy (Shuffle, Round Robin).
3. **The Tunnel**: If enabled, traffic is routed through a SOCKS5 proxy (127.0.0.1:1081) to bypass bandwidth limits.
4. **The Strike**: `Pydantic AI` executes the prompt against the target model (Gemini 3.1, Llama 4, etc.).
5. **The Analysis**: Tokens are counted, cost is calculated, and real-time RPM/TPM meters are updated.
6. **The Vault**: `HighSignalLogger` generates a unique Tag ID, appends to master logs, and saves verbatim I/O to the forensic vault.
7. **The Return**: The app receives the response + Tag ID + Usage metrics.
